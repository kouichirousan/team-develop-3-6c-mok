# 🎮 Office Vibe Connector - デモガイド

## クイックスタート

```bash
# 1. 依存関係をインストール
npm install

# 2. 開発サーバーを起動
npm run dev

# 3. ブラウザで開く
# http://localhost:3000/demo
```

## 🎯 デモで体験できること

### 1. チェックイン/アウト体験

- 巨大な円形ボタンをクリック
- 「Good Morning! ☀️」のメッセージ表示
- ハプティクスフィードバック（対応デバイスのみ）
- リアルタイムでポイント加算

### 2. 早起き鳥ボーナス

**操作手順:**
1. 「🌅 早朝モード」ボタンをクリック（時刻が7:30に変更）
2. チェックインボタンを押す
3. 紙吹雪アニメーションが発動！
4. +30pt獲得（基本10pt + 早朝ボーナス20pt）
5. 2秒後にバッジ獲得モーダルが表示

### 3. オフィスVibeヒートマップ

**シミュレーション:**
- 「🔥 混雑状態」: 42人出社 → オレンジ色に輝く
- 「❄️ 静かな状態」: 8人出社 → クールブルーに変化

色の変化とアニメーションでオフィスの活気を視覚化

### 4. バッジシステム

**インタラクション:**
- 獲得済みバッジをクリック → プロフィールに表示/非表示
- 最大3つまで同時表示可能
- 未獲得バッジは🔒マークでロック表示

**現在獲得済み:**
- ✅ 早起き鳥 (Bronze)
- ✅ 早起き鳥 (Silver)
- ✅ オフィスの守り人

**未獲得:**
- 🔒 早起き鳥 (Gold)
- 🔒 フードファイター

### 5. ランキングシステム

- リアルタイムで順位表示
- 🥇🥈🥉 メダル表示（上位3名）
- 自分の順位は黄色ハイライト
- ポイントとバッジ数を表示

### 6. ラッキーパーソン

- 趣味タグが一致する人をレコメンド
- 現在のステータス（集中/雑談/相談）を表示
- 共通の趣味を明示

### 7. アクティビティログ

すべてのアクションがリアルタイムで記録：
- ✅ チェックイン/アウト
- 🌅 早起き鳥ボーナス
- 🏆 バッジ獲得
- ⏰ 時刻変更
- 👥 オフィス状態変更
- 📌 バッジ表示切替

## 🎨 デザインの特徴

### Neubrutalism
- **太い境界線**: すべての要素に4px〜8pxの黒枠
- **ドロップシャドウ**: オフセットシャドウで立体感
- **鮮やかな色**: エレクトリック・ブルー、ネオン・イエロー
- **フラット＋立体**: 矛盾を楽しむデザイン

### マイクロインタラクション
- **ホバー**: 要素が少し拡大＆回転
- **クリック**: 縮小してから元に戻る
- **アニメーション**: Framer Motionでスムーズな動き
- **紙吹雪**: 特別な瞬間を祝福

## 🔧 技術的な見どころ

### アニメーション
- **Framer Motion**: 宣言的なアニメーション定義
- **AnimatePresence**: 要素の出現/消失をスムーズに
- **Variants**: 複雑なアニメーションシーケンス

### 状態管理
- **React Hooks**: useState, useEffectでシンプルに
- **リアルタイム更新**: 時刻、ポイント、ログ

### レスポンシブデザイン
- **Tailwind CSS**: ユーティリティファーストで高速開発
- **Grid/Flexbox**: 柔軟なレイアウト
- **モバイル対応**: 画面サイズに応じて最適化
